# 🚀 Interactive Navigation Menu

A simple interactive navigation bar built using **HTML, CSS, and JavaScript**.
The navbar stays fixed and changes its style when the user scrolls or hovers over menu items.

---

## ✨ Features

* Fixed navbar (always visible)
* Hover effects on menu items
* Style changes on scroll
* Smooth and clean UI

---

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript
